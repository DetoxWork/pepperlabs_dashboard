PepperLabs Dashboard - Docker Requirements
==========================================

1) Base runtime
- Node.js: 20 LTS recommended
  (Vite 6 requirement: ^18.0.0 || ^20.0.0 || >=22.0.0)
- npm: bundled with Node.js (npm 9+ works, npm 10+ recommended)

2) Application dependencies
- Use lockfile-based install:
  npm ci
- Main packages:
  - vite (dev dependency)
  - chart.js

3) Required environment variables
- VITE_AIRTABLE_TOKEN
- VITE_AIRTABLE_BASE_ID
- VITE_AIRTABLE_TABLE_NAME

4) Optional environment variables
- VITE_AIRTABLE_CONCURRENCY
  - integer range: 1-8
  - default: 6

5) Network requirements
- Outbound HTTPS access to:
  https://api.airtable.com

6) Ports to expose
- 5173 for Vite dev server (npm run dev)
- 4173 for Vite preview server (npm run preview)

7) Container commands
- Development:
  npm ci
  npm run dev -- --host 0.0.0.0 --port 5173

- Production-like preview:
  npm ci
  npm run build
  npm run preview -- --host 0.0.0.0 --port 4173

- Static production hosting option:
  npm ci
  npm run build
  # Serve ./dist with nginx/caddy/http-server in a runtime stage

8) Docker dev note (optional)
- If file watching is unreliable in mounted volumes, set:
  CHOKIDAR_USEPOLLING=true
